# 100 Humans vs Gorilla Simulation

Simulation exploring hypothetical outcomes between humans and a gorilla.

## What it does
- Asks scenario-based questions
- Estimates probabilities of different outcomes
- Generates a descriptive narrative of the scenario

## How it was built
Focuses on logic, scoring, and narrative generation rather than UI.  
Interactive question-answer flow guides the simulation and determines results.

## Tools
- Python
- Optional GUI via Tkinter
- Basic probabilistic modeling

## Status
Functional prototype.
